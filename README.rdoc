== Rtopology

Rtopology is a web crawler that scrapes all pages publshed via http on a given domain name. Rtopology scans pages for links and builds up a database of each published page. This database then allows Rtopology to do the following
 * Highlight SEO issues with a given site
 * Highlight slow to respond pages
 * Record any pages that are respoding with error codes
 * Test if a domain and a given set of pages are responding as they should
